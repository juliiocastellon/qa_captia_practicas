# 💡 Guía de Uso

La categoría **Bloques IA** está diseñada para incorporar capacidades de inteligencia artificial dentro de workflows de automatización.

Se utiliza principalmente para:

- Automatizar decisiones
- Analizar información
- Procesar contexto
- Generar respuestas
- Aplicar lógica inteligente
- Construir automatizaciones avanzadas
- Crear flujos dinámicos

## ✅ Cómo utilizar correctamente los nodos de Bloques IA

1. Expande la categoría “Bloques IA”.
2. Arrastra el nodo deseado al canvas.
3. Configura los parámetros y prompts necesarios.
4. Conecta el nodo con el resto del workflow.
5. Guarda los cambios realizados.
