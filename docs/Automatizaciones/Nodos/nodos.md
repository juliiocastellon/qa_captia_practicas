# 💡 Guía de Uso

La sección **Categorías de Nodos** está diseñada para organizar todos los nodos disponibles dentro del editor de automatizaciones.

Se utiliza principalmente para:

- Organizar nodos
- Localizar herramientas
- Construir workflows
- Gestionar automatizaciones
- Acceder a funcionalidades
- Crear lógica operativa
- Facilitar navegación

## ✅ Cómo utilizar correctamente las categorías de nodos

1. Expande la categoría deseada.
2. Localiza el nodo que quieras utilizar.
3. Arrastra el nodo al canvas del workflow.
4. Configura sus propiedades correspondientes.
5. Conecta el nodo con el resto de la automatización.
6. Guarda los cambios realizados.
