# 🧩 Despiece de Elementos Funcionales

| # | Nombre del Elemento | Tipo | Destino / Acción | Descripción Funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Constructor de Fórmula | Nodo de cálculo | Configura operaciones lógicas | Permite crear fórmulas y operaciones utilizando placeholders, constantes y operadores dentro del workflow. |
| 1.1 | Configuración "Placeholders (entradas)" | Configuración de entradas | Añade variables de entrada | Permite definir las variables o placeholders que serán utilizados dentro de la fórmula. Para utilizarlo, el usuario debe añadir placeholders y seleccionar las entradas deseadas. |
| 1.2 | Campo "Fórmula" | Editor de fórmulas | Construye operaciones | Permite escribir y configurar la operación lógica o matemática que ejecutará el nodo. Para utilizarlo, el usuario debe insertar operadores, placeholders o constantes dentro del editor. |
| 1.3 | Panel de operadores y constantes | Biblioteca de operaciones | Inserta operadores | Permite acceder rápidamente a operadores lógicos, matemáticos y constantes para construir la fórmula. Para utilizarlo, el usuario debe seleccionar el operador o constante que quiera insertar dentro de la expresión. |
| 1.4 | Botón "Probar fórmula" | Acción de validación | Ejecuta prueba | Permite comprobar si la fórmula configurada es válida antes de guardar el nodo. Para utilizarlo, el usuario debe pulsar “Probar fórmula” y revisar el resultado obtenido. |

# 💡 Guía de Uso

El nodo **Fórmula** está diseñado para realizar cálculos y operaciones lógicas dentro de workflows de automatización.

Se utiliza principalmente para:

- Crear operaciones lógicas
- Aplicar cálculos
- Comparar valores
- Transformar datos
- Gestionar expresiones
- Automatizar decisiones
- Construir lógica operativa

## ✅ Cómo configurar correctamente el nodo Fórmula

1. Arrastra el nodo al canvas del workflow.
2. Añade los placeholders necesarios.
3. Configura la fórmula utilizando operadores y constantes.
4. Prueba el resultado utilizando “Probar fórmula”.
5. Guarda los cambios realizados.
6. Conecta el nodo con el resto del workflow.
