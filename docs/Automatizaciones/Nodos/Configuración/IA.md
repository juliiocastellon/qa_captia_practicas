# 🧩 Despiece de Elementos Funcionales

| # | Nombre del Elemento | Tipo | Destino / Acción | Descripción Funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Nodo "Decisión con IA" | Nodo IA | Configura decisiones inteligentes | Permite utilizar inteligencia artificial para analizar información y determinar automáticamente el siguiente paso dentro del workflow. |
| 1.1 | Campo "Nombre del nodo" | Campo de texto | Renombra el nodo | Permite asignar un nombre identificativo al nodo IA dentro del workflow. Para utilizarlo, el usuario debe escribir el nombre deseado dentro del campo correspondiente. |
| 1.2 | Campo "Descripción" | Campo de texto | Añade descripción | Permite añadir información descriptiva sobre el comportamiento o propósito del nodo. Para utilizarlo, el usuario debe escribir el texto descriptivo correspondiente. |
| 1.3 | Campo "Prompt del sistema" | Configuración IA | Define comportamiento del modelo | Permite configurar las instrucciones generales que seguirá el modelo de IA durante la ejecución del workflow. Para utilizarlo, el usuario debe escribir las reglas o contexto que definirán el comportamiento esperado del modelo. |
| 1.4 | Campo "Prompt del usuario" | Configuración IA | Define consulta o contexto | Permite introducir la petición, pregunta o información que será enviada al modelo de IA para generar una respuesta o decisión. Para utilizarlo, el usuario debe escribir el contexto o instrucción correspondiente. |
| 1.5 | Estado "Bloque IA" | Indicador informativo | Muestra disponibilidad | Permite visualizar el estado de disponibilidad del nodo IA dentro de la plataforma. |

# 💡 Guía de Uso

El nodo **Decisión con IA** está diseñado para incorporar capacidades de inteligencia artificial dentro de workflows de automatización.

Se utiliza principalmente para:

- Automatizar decisiones
- Analizar información
- Generar respuestas
- Procesar contexto
- Aplicar lógica inteligente
- Gestionar automatizaciones avanzadas
- Construir flujos dinámicos

## ✅ Cómo configurar correctamente el nodo Decisión con IA

1. Arrastra el nodo al canvas del workflow.
2. Configura el nombre y descripción del nodo.
3. Define las instrucciones del sistema.
4. Configura el prompt o contexto del usuario.
5. Guarda los cambios realizados.
6. Conecta el nodo con el resto del workflow.
