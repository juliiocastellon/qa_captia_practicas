# 💡 Guía de Uso

El nodo **Variable** está diseñado para utilizar referencias y valores dinámicos dentro de workflows de automatización.

Se utiliza principalmente para:

- Utilizar variables
- Compartir datos
- Referenciar valores
- Gestionar información
- Automatizar procesos
- Construir lógica operativa
- Conectar datos entre nodos

## ✅ Cómo configurar correctamente el nodo Variable

1. Arrastra el nodo al canvas del workflow.
2. Busca o selecciona la variable deseada.
3. Configura la referencia correspondiente.
4. Guarda los cambios realizados.
5. Conecta el nodo con el resto del workflow.
