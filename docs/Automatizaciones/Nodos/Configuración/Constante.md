# 🧩 Despiece de Elementos Funcionales

| # | Nombre del Elemento | Tipo | Destino / Acción | Descripción Funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Nodo "Constante" | Nodo de valor fijo | Configura valor constante | Permite utilizar valores fijos dentro del workflow para cálculos, comparaciones o configuraciones lógicas. |
| 1.1 | Campo "Tipo de dato" | Selector desplegable | Selecciona tipo de valor | Permite definir el tipo de dato que utilizará la constante, como número, texto o booleano. Para utilizarlo, el usuario debe seleccionar el tipo deseado dentro del desplegable. |
| 1.2 | Campo "Valor constante" | Campo de entrada | Define valor fijo | Permite introducir el valor constante que utilizará el workflow. Para utilizarlo, el usuario debe escribir el valor correspondiente según el tipo de dato seleccionado. |

---

# 💡 Guía de Uso

El nodo **Constante** está diseñado para utilizar valores fijos dentro de workflows de automatización.

Se utiliza principalmente para:

- Definir valores fijos
- Realizar comparaciones
- Configurar parámetros
- Aplicar cálculos
- Automatizar decisiones
- Gestionar lógica operativa
- Compartir datos entre nodos

## ✅ Cómo configurar correctamente el nodo Constante

1. Arrastra el nodo al canvas del workflow.
2. Selecciona el tipo de dato correspondiente.
3. Introduce el valor constante deseado.
4. Guarda los cambios realizados.
5. Conecta el nodo con el resto del workflow.
