# 🖥️ Crear evento

**Objetivo principal:** Permite al usuario crear y programar eventos dentro del calendario, definiendo tipo, categoría, fechas, responsables y contexto operativo asociado.

## 🧩 Despiece de elementos funcionales

| # | Nombre del elemento | Tipo | Destino / Acción | Descripción funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Campo "Título" | Campo de texto | Introducir título | Permite definir el nombre principal del evento que aparecerá en el calendario. |
| 2 | Campo "Descripción" | Campo de texto | Añadir descripción | Permite introducir información adicional o detalles relacionados con el evento. |
| 3 | Selector de tipo de evento | Selector desplegable | Cambiar tipo | Permite seleccionar el tipo de evento que se va a crear dentro del calendario. |
| 4 | Selector de categoría | Selector visual | Elegir categoría | Permite asignar una categoría visual al evento, como recordatorio, mantenimiento u otros tipos disponibles. |
| 5 | Configuración de fecha y hora | Selector temporal | Programar evento | Permite definir fechas y horas de inicio y fin del evento, incluyendo opción de evento de día completo. |
| 6 | Campo "Asignado a" | Selector de usuario | Asignar responsable | Permite seleccionar el usuario responsable o asociado al evento creado. |
| 7 | Campo "Planta" | Selector contextual | Asociar planta | Permite vincular el evento a una planta o instalación concreta. |
| 8 | Campo "Zona" | Selector contextual | Asociar zona | Permite asociar el evento a una zona específica dentro de la instalación. |
| 9 | Campo "Máquina" | Selector contextual | Asociar máquina | Permite vincular el evento a una máquina o dispositivo concreto. |
| 10 | Gestión de recordatorios | Configuración adicional | Añadir recordatorios | Permite configurar avisos y recordatorios automáticos relacionados con el evento. |

## 💡 Guía de uso

La ventana de creación de eventos permite programar actividades, mantenimientos, alarmas o recordatorios dentro del calendario operativo. Para crear un evento, primero debes definir un título y, opcionalmente, añadir una descripción más detallada.

Posteriormente, puedes seleccionar el tipo y categoría del evento, establecer las fechas y horas correspondientes y asociarlo a usuarios, plantas, zonas o máquinas específicas. También es posible configurar recordatorios automáticos para asegurar el seguimiento adecuado de la actividad programada.
