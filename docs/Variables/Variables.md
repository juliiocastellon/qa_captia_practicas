# 🖥️ Variables

**Objetivo principal:** Permite al usuario visualizar, gestionar y filtrar las variables y señales disponibles dentro de la plataforma para su utilización en dashboards, automatizaciones, SCADA e informes.

## 🧩 Despiece de elementos funcionales

| # | Nombre del elemento | Tipo | Destino / Acción | Descripción funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Pestaña "Variables" | Navegación por pestañas | Cambiar vista | Permite al usuario acceder al listado general de variables registradas en la plataforma. Para utilizarla, el usuario debe pulsar sobre la pestaña “Variables” para visualizar y gestionar todas las variables disponibles. |
| 2 | Pestaña "Realtime" | Navegación por pestañas | Ver datos realtime | Permite al usuario visualizar variables y señales actualizadas en tiempo real. Para utilizarla, el usuario debe acceder a la pestaña “Realtime” y podrá supervisar datos que se actualizan automáticamente. |
| 3 | Botón "Nueva" | CTA Principal | Crear variable | Permite al usuario crear una nueva variable dentro de la plataforma. Para utilizarlo, el usuario debe pulsar el botón “Nueva”, completar la configuración requerida y guardar la nueva variable. |
| 4 | Botón "Sincronizar" | CTA Secundario | Actualizar variables | Permite al usuario actualizar y sincronizar variables obtenidas desde conectores o sistemas externos. Para utilizarlo, el usuario debe pulsar el botón “Sincronizar” para refrescar la información disponible. |
| 5 | Indicadores de estado de variables | Indicadores informativos | — | Permite al usuario visualizar estadísticas rápidas sobre variables encontradas, activas o descubiertas dentro del sistema para conocer el estado general de la monitorización. |
| 6 | Buscador de variables | Campo de búsqueda | Filtrar variables | Permite al usuario localizar variables rápidamente mediante su nombre visible o técnico. Para utilizarlo, el usuario debe escribir el nombre deseado dentro del buscador y el listado se filtrará automáticamente. |
| 7 | Filtro de estado | Filtro desplegable | Filtrar variables | Permite al usuario mostrar variables según su estado operativo o disponibilidad. Para utilizarlo, el usuario debe seleccionar el estado deseado desde el desplegable correspondiente. |
| 8 | Filtro de origen | Filtro desplegable | Filtrar por origen | Permite al usuario visualizar variables según la fuente o sistema de origen asociado. Para utilizarlo, el usuario debe seleccionar el origen deseado desde el filtro disponible. |
| 9 | Filtro de categoría | Filtro desplegable | Filtrar por categoría | Permite al usuario organizar las variables según categorías funcionales configuradas. Para utilizarlo, el usuario debe seleccionar una categoría desde el desplegable para filtrar el listado mostrado. |
| 10 | Filtro de fábrica | Filtro desplegable | Filtrar por fábrica | Permite al usuario limitar las variables mostradas a una fábrica o instalación concreta. Para utilizarlo, el usuario debe seleccionar la fábrica deseada desde el filtro disponible. |
| 11 | Contador de resultados | Indicador informativo | — | Permite al usuario visualizar el número total de variables que coinciden actualmente con los filtros aplicados dentro del listado. |
| 12 | Listado de variables | Tabla de datos | Gestionar variables | Permite al usuario visualizar y gestionar todas las variables disponibles mediante una tabla detallada. Para utilizarlo, el usuario puede consultar información técnica, estado, origen o acceder a opciones de configuración y edición de cada variable. |

## 💡 Guía de uso

La sección de Variables centraliza todas las señales y variables disponibles dentro de la plataforma. Desde esta vista puedes buscar variables concretas, filtrarlas por estado, categoría, origen o fábrica y consultar rápidamente información técnica relevante.

La tabla principal muestra datos detallados de cada variable, incluyendo nombre técnico, unidad, tipo, categoría y estado operativo. Además, es posible sincronizar señales externas, crear nuevas variables y alternar entre una vista general y una vista realtime para supervisar cambios en tiempo real.
