# 🖥️ Variables

**Objetivo principal:** Permite al usuario visualizar, gestionar y filtrar las variables y señales disponibles dentro de la plataforma para su utilización en dashboards, automatizaciones, SCADA e informes.

## 🧩 Despiece de elementos funcionales

| # | Nombre del elemento | Tipo | Destino / Acción | Descripción funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Pestaña "Variables" | Navegación por pestañas | Cambiar vista | Permite acceder al listado general de variables registradas en la plataforma. |
| 2 | Pestaña "Realtime" | Navegación por pestañas | Ver datos realtime | Permite visualizar variables y señales actualizadas en tiempo real. |
| 3 | Botón "Nueva" | CTA Principal | Crear variable | Permite crear y registrar una nueva variable dentro de la plataforma. |
| 4 | Botón "Sincronizar" | CTA Secundario | Actualizar variables | Permite sincronizar y refrescar las variables obtenidas desde fuentes externas o conectores industriales. |
| 5 | Indicadores de estado de variables | Indicadores informativos | — | Muestra estadísticas rápidas sobre el número de variables encontradas, activas y descubiertas. |
| 6 | Buscador de variables | Campo de búsqueda | Filtrar variables | Permite localizar variables rápidamente mediante nombre visible o nombre técnico. |
| 7 | Filtro de estado | Filtro desplegable | Filtrar variables | Permite mostrar variables según su estado operativo o disponibilidad. |
| 8 | Filtro de origen | Filtro desplegable | Filtrar por origen | Permite visualizar variables según la fuente o sistema de origen asociado. |
| 9 | Filtro de categoría | Filtro desplegable | Filtrar por categoría | Permite filtrar variables según categorías funcionales configuradas. |
| 10 | Filtro de fábrica | Filtro desplegable | Filtrar por fábrica | Permite limitar las variables mostradas a una fábrica o instalación concreta. |
| 11 | Contador de resultados | Indicador informativo | — | Muestra el número de variables que coinciden actualmente con los filtros aplicados. |
| 12 | Listado de variables | Tabla de datos | Gestionar variables | Muestra todas las variables disponibles mediante una tabla detallada con información técnica, estado, origen, unidad y fecha de actualización. |

## 💡 Guía de uso

La sección de Variables centraliza todas las señales y variables disponibles dentro de la plataforma. Desde esta vista puedes buscar variables concretas, filtrarlas por estado, categoría, origen o fábrica y consultar rápidamente información técnica relevante.

La tabla principal muestra datos detallados de cada variable, incluyendo nombre técnico, unidad, tipo, categoría y estado operativo. Además, es posible sincronizar señales externas, crear nuevas variables y alternar entre una vista general y una vista realtime para supervisar cambios en tiempo real.
