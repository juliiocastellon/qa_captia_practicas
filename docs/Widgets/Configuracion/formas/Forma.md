# 🧩 Despiece de Elementos Funcionales

| # | Nombre del Elemento | Tipo | Destino / Acción | Descripción Funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Pestaña "Datos" | Configuración básica | Configura comportamiento visual | Permite acceder a las configuraciones relacionadas con la forma y sus reglas dinámicas. |
| 1.1 | Campo "Nombre del widget" | Campo de texto | Renombra el widget | Permite definir el nombre identificativo del widget dentro del dashboard o vista SCADA. Para utilizarlo, el usuario debe escribir el nombre que quiera mostrar en la cabecera del widget. |
| 1.2 | Sección "Configuración de Forma" | Configuración visual | Configura la geometría visual | Permite seleccionar el tipo de forma y personalizar sus colores básicos. Para utilizarlo, el usuario debe elegir la geometría deseada y configurar colores de relleno y borde. |
| 1.3 | Sección "Configuración de Imagen" | Configuración multimedia | Añade imágenes a la forma | Permite utilizar imágenes adaptadas a la geometría de la forma. Para utilizarlo, el usuario debe activar la opción correspondiente para aplicar imágenes sobre la forma visual. |
| 1.4 | Sección "Etiqueta" | Configuración textual | Configura etiquetas visibles | Permite añadir texto o etiquetas descriptivas asociadas a la forma. Para utilizarlo, el usuario debe introducir el contenido textual que quiera mostrar junto al elemento visual. |

| 2.1 | Sección "Contenedor" | Configuración visual | Ajusta el aspecto externo | Permite modificar el fondo, transparencia y estilo visual del contenedor. Para utilizarlo, el usuario debe seleccionar el estilo deseado y ajustar colores y opacidad. |
| 2.2 | Sección "Propiedades de la Forma" | Configuración visual | Ajusta geometría y visualización | Permite modificar grosor de borde, opacidad y radio de esquinas de la forma. Para utilizarlo, el usuario debe mover los sliders hasta conseguir la visualización deseada. |
| 3 | Pestaña "Texto" | Configuración textual | Configura textos auxiliares | Permite acceder a configuraciones relacionadas con textos adicionales mostrados junto a la forma. |

# 💡 Guía de Uso

El widget **Forma** está diseñado para representar visualmente estados, condiciones y comportamientos dentro de dashboards y pantallas SCADA.

Se utiliza principalmente para:

- Representar estados visuales
- Mostrar condiciones dinámicas
- Crear indicadores gráficos
- Visualizar procesos
- Añadir elementos SCADA visuales
- Mostrar comportamiento en tiempo real
- Complementar diagramas y vistas industriales

## ✅ Cómo configurar correctamente el widget Forma

1. Arrastra el widget al canvas.
2. Selecciona el widget dentro del dashboard o SCADA.
3. Desde el panel de propiedades:
   - Configura la geometría de la forma.
   - Ajusta colores y bordes.
   - Configura reglas dinámicas.
   - Selecciona variables de referencia.
   - Personaliza opacidad y apariencia.
   - Añade etiquetas y textos adicionales.
4. Guarda los cambios realizados.
