# 🧩 Despiece de Elementos Funcionales

| # | Nombre del Elemento | Tipo | Destino / Acción | Descripción Funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Pestaña "Datos" | Configuración básica | Configura información general | Permite acceder a las configuraciones generales del widget de texto. |
| 1.1 | Campo "Nombre del widget" | Campo de texto | Renombra el widget | Permite definir el nombre identificativo del widget dentro del dashboard. Para utilizarlo, el usuario debe escribir el nombre que quiera mostrar en la cabecera del widget. |
| 2 | Pestaña "Apariencia" | Configuración visual | Personaliza el contenedor | Permite acceder a las configuraciones visuales relacionadas con el aspecto externo del widget. |

| 3 | Pestaña "Texto" | Configuración textual | Configura el contenido mostrado | Permite acceder a las configuraciones relacionadas con el contenido textual del widget. |
| 3.1 | Sección "Posición del texto" | Configuración de alineación | Ajusta posición vertical | Permite definir la posición vertical del texto dentro del widget. Para utilizarlo, el usuario debe seleccionar si quiere mostrar el texto arriba, centrado o abajo. |
| 3.2 | Sección "Contenido de texto" | Editor textual | Edita el contenido mostrado | Permite escribir y modificar el contenido textual del widget. Para utilizarlo, el usuario debe pulsar “Editar texto” para abrir el editor enriquecido. |

| 3.2.2 | Herramientas de formato rápido | Formato de texto | Cambia estilos tipográficos | Permite aplicar estilos rápidos como negrita, cursiva o subrayado al contenido textual seleccionado. |
| 3.2.3 | Herramienta "Color de texto" | Selector visual | Cambia color del texto | Permite modificar el color del contenido textual mostrado dentro del widget. |
| 3.2.4 | Selector de tamaño | Selector desplegable | Ajusta tamaño tipográfico | Permite modificar el tamaño visual del texto seleccionado. |
| 3.2.5 | Selector de fuente | Selector desplegable | Cambia familia tipográfica | Permite seleccionar la tipografía utilizada dentro del contenido textual. |
| 3.2.6 | Herramientas de encabezado | Estilo estructural | Cambia jerarquía visual | Permite aplicar estilos de encabezado como H1, H2 o H3 al texto seleccionado. |
| 3.2.7 | Herramientas de alineación | Configuración visual | Ajusta alineación horizontal | Permite alinear el contenido textual a izquierda, centro o derecha. |
| 3.2.8 | Herramientas de espaciado | Configuración visual | Ajusta distribución del texto | Permite modificar la separación y distribución visual del contenido textual. |

# 💡 Guía de Uso

El widget **Texto** está diseñado para añadir contenido textual estático dentro de dashboards y paneles Analytics.

Se utiliza principalmente para:

- Añadir títulos
- Crear encabezados
- Mostrar etiquetas
- Añadir descripciones
- Incorporar notas informativas
- Organizar secciones visuales
- Complementar dashboards con texto

## ✅ Cómo configurar correctamente el widget Texto

1. Arrastra el widget al canvas.
2. Selecciona el widget dentro del dashboard.
3. Desde el panel de propiedades:
   - Configura el nombre del widget.
   - Personaliza el contenedor visual.
   - Ajusta la posición del texto.
   - Edita el contenido textual.
   - Configura colores, tamaños y estilos.
4. Guarda los cambios realizados.
