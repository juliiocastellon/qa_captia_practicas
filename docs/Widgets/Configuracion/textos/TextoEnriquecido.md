# 💡 Guía de Uso

El widget **Texto Enriquecido** está diseñado para añadir contenido textual avanzado y visualmente estructurado dentro de dashboards y paneles Analytics.

Se utiliza principalmente para:

- Crear contenido visual avanzado
- Añadir documentación contextual
- Mostrar instrucciones visuales
- Incorporar contenido estructurado
- Crear bloques informativos
- Añadir contenido dinámico
- Complementar dashboards con formato avanzado

## ✅ Cómo configurar correctamente el widget Texto Enriquecido

1. Arrastra el widget al canvas.
2. Selecciona el widget dentro del dashboard.
3. Desde el panel de propiedades:
   - Configura el nombre del widget.
   - Personaliza el contenedor visual.
   - Ajusta la posición del texto.
   - Edita el contenido enriquecido.
   - Configura tamaños, colores y estilos.
   - Organiza el contenido utilizando encabezados y listas.
4. Guarda los cambios realizados.
