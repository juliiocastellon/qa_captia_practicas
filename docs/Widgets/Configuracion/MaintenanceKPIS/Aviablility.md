# 🧩 Despiece de Elementos Funcionales

| # | Nombre del Elemento | Tipo | Destino / Acción | Descripción Funcional |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Pestaña "Datos" | Configuración de datos | Configura información del KPI | Permite acceder a las configuraciones relacionadas con los datos y el rango temporal del indicador. |
| 1.1 | Campo "Nombre del widget" | Campo de texto | Renombra el widget | Permite definir el nombre identificativo del KPI dentro del dashboard. Para utilizarlo, el usuario debe escribir el nombre que quiera mostrar en la cabecera del widget. |
| 1.2 | Sección "Configuración General" | Configuración de datos | Ajusta agregación e intervalo | Permite configurar cómo se agruparán y visualizarán los datos del KPI. Para utilizarlo, el usuario debe seleccionar el tipo de agregación y el intervalo temporal deseado. |
| 1.3 | Sección "Rango de Tiempo" | Configuración temporal | Configura el periodo mostrado | Permite definir el rango temporal utilizado por el widget. Para utilizarlo, el usuario puede utilizar el filtro global o configurar un periodo específico. |
| 2 | Pestaña "Apariencia" | Configuración visual | Personaliza el widget | Permite acceder a las configuraciones visuales relacionadas con el aspecto externo del KPI. |
| 2.1 | Sección "Contenedor" | Configuración visual | Ajusta el aspecto del widget | Permite modificar el fondo, transparencia y estilo visual del contenedor. Para utilizarlo, el usuario debe seleccionar el estilo deseado y ajustar colores y opacidad. |
| 3 | Pestaña "Texto" | Configuración textual | Configura textos auxiliares | Permite acceder a configuraciones relacionadas con textos adicionales mostrados dentro del widget. |
| 3.1 | Sección "Textos adicionales" | Configuración textual | Añade información contextual | Permite añadir subtítulos y notas adicionales dentro del KPI. Para utilizarlo, el usuario debe rellenar los campos disponibles con el texto que quiera mostrar. |
| 3.1.1 | Campo "Subtítulo" | Campo de texto | Añade texto secundario | Permite mostrar un texto descriptivo debajo del título principal del KPI. |
| 3.1.2 | Campo "Nota al pie" | Campo de texto | Añade información adicional | Permite mostrar información adicional o aclaraciones en la parte inferior del widget. |

# 💡 Guía de Uso

El widget **Availability** está diseñado para visualizar indicadores relacionados con disponibilidad y estado operativo dentro de dashboards Analytics.

Se utiliza principalmente para:

- Supervisar disponibilidad
- Visualizar estados operativos
- Analizar rendimiento temporal
- Monitorizar indicadores
- Comparar periodos
- Facilitar análisis visual
- Mostrar métricas operativas

## ✅ Cómo configurar correctamente el widget Availability

1. Arrastra el widget al canvas.
2. Selecciona el widget dentro del dashboard.
3. Desde el panel de propiedades:
   - Configura el nombre del KPI.
   - Ajusta agregaciones e intervalos.
   - Configura el rango temporal.
   - Personaliza el aspecto visual.
   - Añade textos adicionales si es necesario.
4. Guarda los cambios realizados.
