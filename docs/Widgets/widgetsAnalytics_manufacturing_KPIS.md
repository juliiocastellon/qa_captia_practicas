# 🖥️ Manufacturing KPIs

**Objetivo Principal:** Permite al usuario seleccionar e insertar indicadores de rendimiento industrial específicos de fabricación para monitorizar la eficiencia, calidad y estado operativo de sus máquinas y líneas de producción.

## 🧩 Despiece de Elementos Funcionales

| # | Elemento | Tipo | Acción Principal | Descripción Funcional |
|---|---|---|---|---|
| 3.4 | Sección "Manufacturing KPIs" | Grupo / Acordeón | Expandir o colapsar bloque | Despliega el bloque para acceder a los indicadores industriales específicos de fabricación disponibles. |
| 3.4.1 | OEE | Elemento seleccionable | Insertar indicador OEE | Arrastra el widget a la plantilla o haz clic para insertarlo cuando necesites visualizar la eficiencia global del equipo en un único indicador. |
| 3.4.2 | Disponibilidad | Elemento seleccionable | Insertar indicador Availability | Utilízalo para representar el porcentaje de tiempo operativo real del activo. Puedes arrastrarlo a la plantilla o insertarlo directamente con un clic. |
| 3.4.3 | Rendimiento | Elemento seleccionable | Insertar indicador Performance | Inserta este widget en la plantilla para monitorizar si la velocidad de producción real coincide con la velocidad esperada. |
| 3.4.4 | Calidad | Elemento seleccionable | Insertar indicador Quality | Arrástralo al dashboard para visualizar el porcentaje de producción conforme respecto al total fabricado. |
| 3.4.5 | Unidades Buenas | Elemento seleccionable | Insertar contador Good Units | Añade este indicador a la plantilla cuando necesites contabilizar las unidades válidas producidas durante un periodo. |
| 3.4.6 | Tasa de Scrap | Elemento seleccionable | Insertar indicador Scrap Rate | Utiliza este widget para controlar el porcentaje de rechazo o desperdicio generado durante la producción. |
| 3.4.7 | Tiempo de Paro | Elemento seleccionable | Insertar indicador Downtime | Inserta el componente en el dashboard para analizar el tiempo perdido por paradas de máquina o incidencias. |
| 3.4.8 | Energía | Elemento seleccionable | Insertar indicador Energy | Arrastra este widget a la plantilla para supervisar el consumo energético del activo seleccionado. |
| 3.4.9 | Throughput | Elemento seleccionable | Insertar indicador de producción | Utilízalo para representar la cantidad de unidades producidas por hora o por intervalo definido. |
| 3.4.10 | OEE Trend | Elemento seleccionable | Insertar gráfico histórico de OEE | Añade este gráfico al dashboard para analizar la evolución temporal del OEE y detectar tendencias. |
| 3.4.11 | Timeline de Estados | Elemento seleccionable | Insertar timeline de estados | Inserta este componente cuando necesites visualizar cronológicamente los estados operativos de las máquinas. |
| 3.4.12 | Ranking Máquinas | Elemento seleccionable | Insertar ranking por KPIs | Utiliza este widget para comparar el rendimiento entre activos y detectar rápidamente los equipos más eficientes o críticos. |

## 💡 Guía de Uso 

La sección **Manufacturing KPIs** agrupa los 12 indicadores industriales esenciales para el seguimiento del rendimiento de tus equipos y líneas de producción. Puedes comenzar con los indicadores fundamentales del OEE —**OEE** (3.4.1), **Disponibilidad** (3.4.2), **Rendimiento** (3.4.3) y **Calidad** (3.4.4)— para obtener una visión global de la eficiencia operativa, y complementarlos con métricas específicas como **Tasa de Scrap** (3.4.6), **Tiempo de Paro** (3.4.7) o **Energía** (3.4.8) según las necesidades de tu análisis. Para una visión más analítica y comparativa, dispones de **OEE Trend** (3.4.10) para identificar patrones temporales, **Timeline Estados** (3.4.11) para auditar el historial de estados de máquina en múltiples activos, y **Ranking Máquinas** (3.4.12) para comparar el desempeño entre equipos de un vistazo.
